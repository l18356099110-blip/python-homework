项目结构
项目根目录/
├─ week6_01.py          # 主程序
├─ .env                 # 本地配置（不上传 GitHub）
├─ task.log             # 运行日志
├─ requirements.txt     # 依赖清单
├─ .gitignore           # Git 忽略规则
└─ README.md            # 说明文档

新建 .env 文件
在项目根目录下新建 .env 文件，内容如下：
CLIENT_ID=你的飞书应用ID
CLIENT_SECRET=你的飞书应用密钥
APP_TOKEN=你的多维表格APP_TOKEN
TABLE_ID=你的表格ID
